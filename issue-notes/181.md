# issue PR 179に関連して、cat2151のライブラリをベンダリングするようCIワークフローファイルを作成する #181
[issues #181](https://github.com/cat2151/tonejs-mml-to-json/issues/181)

- userがほしいものは、毎日重要なバグ修正のあるcat2151のライブラリを、毎日CIでベンダリングしてcommit pushすることである

- このCIは、1日1回、JST 01:00 に実行され、upstreamのcat2151のライブラリが変更されていたらベンダリングしてcommit pushする
- これにより、cat2151のライブラリ（毎日重要なバグ変更があるため、毎日取り込みがマスト）の毎日の最新化が実現される

- 実現方法は、cat2151の他のリポジトリで既に実施済みなので、それを参考にすること
