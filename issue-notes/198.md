# issue ランダム音色のオシレータが、接頭辞 fat / am / fm / pwm を生成していないので、生成させる #198
[issues #198](https://github.com/cat2151/tonejs-mml-to-json/issues/198)

- sawtooth などにくわえ、接頭辞 fat などをつけた fatsawtooth などもランダムで選ぶようにすること
- 接頭辞は、fat / am / fm / pwm / 接頭辞なし の5パターンとすること
- オシレータ内の引数に、接頭辞にあわせたものを追加すること
    - 例えば、oscillator に、sawtooth のかわりに、接頭辞 fatをつけた fatsawtooth をランダム指定した場合、以下の挙動とすること：
      - oscillator の引数に、「count: 3」「spread: 30」などを追加すること。
      - なぜなら、instrument の oscillator に fatsawtooth を指定したということは、
        - Instrument が紐付く Source に FatOscillator を選び、
          - その FatOscillator の oscillator に sawtooth を選んだ、ということなので。
      - ドキュメントを読むべし : https://tonejs.github.io/docs/15.0.4/classes/FatOscillator.html#count
    - ほか、Tone.js v15のドキュメントを調査せよ。わからないならuserに質問せよ
