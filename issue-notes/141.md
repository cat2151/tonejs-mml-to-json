# 補足
- 「当リポジトリをnpm github で（cloneでなく）ライブラリ利用する手順」としてのミニマムなdemoを実装する。
- これもdeploy対象とする。
- これどおりにbuildすればライブラリ利用できる、を実証する用。
- cloneあるいは直接ソース参照ではなく、あくまでnpm github経由でのビルド、という手順、を示すことに注意
    - でないと外部リポジトリからの利用者にとっては意味がない
    - cloneによる利用は手順が複雑である。npmライブラリとしての利用がスタンダードである
