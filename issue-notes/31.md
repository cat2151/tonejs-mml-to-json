# issue エラーのためJSON変換ができない #31
[issues #31](https://github.com/cat2151/tonejs-mml-to-json/issues/31)

## 問題の詳細

GitHub Pages上のデモサイトで以下のエラーが発生していました：

```
GET https://cat2151.github.io/tonejs-mml-to-json/dist/play.js net::ERR_ABORTED 404 (Not Found)
GET https://cat2151.github.io/tonejs-mml-to-json/dist/mml2json-wasm.js net::ERR_ABORTED 404 (Not Found)
GET https://cat2151.github.io/tonejs-mml-to-json/dist/main.js net::ERR_ABORTED 404 (Not Found)
```

## 原因

1. `src/index.html`は`../dist/`ディレクトリからJavaScriptファイルを読み込もうとしていた
2. `dist/`ディレクトリと`pkg/`ディレクトリは`.gitignore`に含まれているため、リポジトリにコミットされていない
3. これらのファイルはTypeScriptとRust/WASMソースからビルドする必要がある
4. GitHub Pagesがリポジトリから直接配信していたため、ビルドステップがなかった

## 解決策

`.github/workflows/deploy-pages.yml`を作成し、以下を実行するGitHub Actionsワークフローを追加しました：

1. Node.jsとRustツールチェーンのセットアップ
2. wasm-packのインストール
3. Rustソースからのwasm-pack build
4. TypeScriptソースからJavaScriptへのビルド
5. ビルドされたファイルを含む完全なサイトをGitHub Pagesにデプロイ

## 注意事項

- このワークフローは`main`ブランチへのpush時に自動的に実行されます
- 手動でワークフローを実行することもできます（workflow_dispatch）
- GitHub PagesはGitHub Actionsを使用してデプロイするように設定する必要があります
