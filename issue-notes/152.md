# Issue #152: Unify loopEnd timing format with triggerAttackRelease

**Status**: ✅ Implemented (BREAKING CHANGE)

## Summary
Changed loopEnd format from `"288i"` to `"+288i"` for consistency with triggerAttackRelease.

## Breaking Change
⚠️ External consumers parsing `loopEnd.args[0]` must handle the `+` prefix.

See IMPLEMENTATION_NOTES_152.md for migration guide and details.
