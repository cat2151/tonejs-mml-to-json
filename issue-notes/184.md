# issue ランダム音色生成関数の機能が想定と違う #184
[issues #184](https://github.com/cat2151/tonejs-mml-to-json/issues/184)

- userが外部project tonejs-step-sequencer でほしいもの
- 「ランダムinstrument+effect」関数と、そのボタン：
    - ランダムinstrumentと、ランダムeffectを選ぶ
    - それぞれの値は、省略されない、すべての値である
    - すべての値を、ランダム音色生成設定用JSONに従ったルールで、ランダムにする
    - その用途は、tonejs-step-sequencer での、以前のランダム音色ボタンが、その機能だったので、それと同じ動作を実現する用

- これまでの課題
  - demoにおける、
  - ランダムinstrumentボタンも、ランダムeffectボタンも、
  - 「省略された代表的な値」しか出力されておらず、userがより高度な編集をしたいニーズがあるのに、ニーズを満たしていない

  - instrumentにおいては、選ぶvoiceやオシレータといったものも、複数の種類から、選べるはず。例えばsawtoothやsquareなどを選べるはず
     - それを選べていないように見受けられる

- 備忘、以前の tonejs-step-sequencer は部分的なinstrumentとeffectの組み合わせを使っていた。今回は検証用に全部の組み合わせでまず実装とする
